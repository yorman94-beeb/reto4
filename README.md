# Ejemplo de programa generar numero aleatorio o fijo

## numero aleatorio
para esste caso es importante conocer como funciona la estructura de repetcion Math.random() acompañada de Math.floor, "Math.floor(Math.random() * (máximo - mínimo + 1) + mínimo)" y el como usar esta para generar
numero aleatorio que este dentro de el rango de numeros que queremos utilizar.

## numero fijo
en este caso el diseñador del programa define el numero, lo que en este caso es importante resaltar es la inportancia del bucle for "for(let contador= 0; contador< 3; contador++)" ya que este 
nos indica el numero de intentos que vamos a permitir y el "++" nos ayuda con ese  incremento en 1, en cuanto a la cantidad de intentos se refiere.
